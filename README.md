# PHP MVC Framework

<p>This framework was not supposed to be used in production, it's just for educational purpose.</p>

## Features
- Custom routing.
- Controllers.
- Models.
- Migrations
- Sessions.
- Middlewares.

## Prerequisites
- Docker

## Installing
    - Build docker image:
        - `docker build ./docker -t phpframework/stable:1.0`
    - Run app with docker compose:
        - `docker compose up -d`
